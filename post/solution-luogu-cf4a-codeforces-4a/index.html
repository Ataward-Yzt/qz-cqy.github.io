题解 CF4A@洛谷 | 4A@Codeforces 【Watermelon】

2019-01-31

首先声明：本题解使用Python3语言

首先，我在纸上列出了一些正整数，发现

所有大于等于4的偶数都可以分成两个正偶数的和

然后，按照以上规律，轻而易举地编写Python3程序（C++有了，只好用Python3）：

n = int(input())
if ((n >= 4) and (n % 2 == 0)):
    print("YES")
else:
    print("NO")
