头文件

哦，对了，这里是C++党领地

主程序

观察一下，答案每次要么加1，要么加2，所以只需要先加上1，然后如果输入的数（a）大于k，则再加1即可。

if(a > k) c++;
c++;

因为有n次循环，所以把上面的放入循环中：

while(n--)
{
	cin>>a;
    if(a > k) c++;
    c++'
}

在前面加上输入和输出，得到程序：

#include <iostream>
using namespace std;

int main()
{
	int n, k;
	cin>>n>>k;
	int a;
	int c = 0;
	while(n--)
	{
		cin>>a;
		if(a > k) c++;
		c++;
	}
	cout<<c<<endl;
	return 0;
}
